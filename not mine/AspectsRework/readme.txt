:: REQUIREMENTS ::

- ModUtil
- ModImporter
- ModImporter Optional SJSON Module
- Up-to-date version of Hades
- Python 3.8 or later

:: INSTALLATION ::

1. Go to your Hades folder.
2. Go to the Content subfolder in the Hades folder.
3. Create a folder named Mods inside the Content folder, if it is not there already.
4. Extract the .zip mod file to the Mods folder.
5. Run ModImporter.
